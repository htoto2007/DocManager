DoubleClick Double clicking a subitem or the primary column will edit that cell. F2 will edit the primary column.
F2Only Pressing F2 is the only way to edit the cells. Once the primary column is being edited, the other cells in the row can be edited by pressing Tab.
None This list cannot be edited. F2 does nothing.
SingleClick A single click on a <strong>subitem</strong> will edit the value. Single clicking the primary column, selects the row just like normal. The user must press F2 to edit the primary column.
