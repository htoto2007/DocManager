Cancel Change this to true to cancel the cell editing operation.
Control During the CellEditStarting event, this can be modified to be the control that you want to edit the value. You must fully configure the control before returning from the event, including its bounds and the value it is showing. During the CellEditFinishing event, you can use this to get the value that the user entered and commit that value to the model. Changing the control during the finishing event has no effect.
