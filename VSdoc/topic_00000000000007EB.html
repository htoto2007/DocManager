Remarks

This designer removes properties and events that are available on ListView but that are not useful on ObjectListView.

We can't inherit from System.Windows.Forms.Design.ListViewDesigner, since it is marked internal. So, this class uses reflection to create a ListViewDesigner and then forwards messages to that designer.
